# Inventory System

## Description

This project is a inventory system written in Python.  
It is a console-based program that collects information about a product and calculates its total cost.

The program asks the user for the product name, price, and quantity.  
It validates the numeric inputs to ensure that the user enters correct values.  
The system calculates the total cost by multiplying the price by the quantity.

While the program runs, it is processing user input and validating the data before performing the final calculation.

## How it works

1. The program asks the user to enter the name of a product.
2. It requests the price of the product and validates that the value is a valid number.
3. The program asks for the quantity and checks that the value is a valid integer.
4. If the user enters invalid data, the program displays an error message and asks for the value again.
5. After the inputs are valid, the program calculates the total cost using the price and quantity.
6. Finally, it prints the product information and the calculated total cost.
